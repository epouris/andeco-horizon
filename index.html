Data Backup & Restore

Export or import all app data (accounting, fleet, crew) to a single file.

⚠️ Import replaces existing data. Export a backup first.

💡 Share data without a server: Put the app folder in a cloud drive (OneDrive, Dropbox, etc.). Each user clicks Use shared data file and selects andeco_data.json in that folder. (Works in Chrome/Edge. In other browsers, use Import Data and Save data to file or Export Data.)

⚠️ Two users at once: The app does not lock the file. If two people edit and save at the same time, the last save overwrites the other. Avoid editing together, or refresh from file before you start and save quickly so others can refresh after you.
